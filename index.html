Abstracción

Diseño de una red sensorial distribuida usando ESP32. Implementación de telemetría en tiempo real sobre protocolos seguros (SSL) y almacenamiento NoSQL en la nube de Firebase.

Arquitectura Stack

{ "core": "Espressif32",
"cloud": "Firebase-RTDB",
"protocol": "HTTPS/WSS" }

Instrumentación

DHT22 (Digital)
V-Anem (Analógico)
±0.5°C / ±2% RH
< 150ms

Dashboard Live

Acceso interactivo a gráficas y registros históricos mediante el código QR.

HARDWARE V1.0

Metodología

Manufactura mediante impresión 3D (FDM) en material PLA+ de alta resistencia térmica. Diseño optimizado para flujo laminar en sensores de viento.

Diagrama Eléctrico

Conexiones blindadas I2C y OneWire.

SENSORS → ESP32 → WIFI → FIREBASE → APP

Conclusiones

El sistema demuestra que la integración de microcontroladores de bajo costo con servicios cloud "serverless" es viable para aplicaciones meteorológicas de precisión.
